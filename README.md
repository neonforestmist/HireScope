# HireScope
An AI-powered web app that analyzes a GitHub portfolio and returns quality feedback.

# How to Install
1. Clone this repository

2. Install Dependencies

```bash
npm install
```
3. Configure environment variables in an `.env` file:
```bash
GEMINI_API_KEY=your_gemini_api_key_here
GEMINI_MODEL=gemini-2.5-flash-lite
PORT=3000
```
4. Start the app:
```bash
npm start
```
5. Open it.

```text
http://localhost:3000
```
